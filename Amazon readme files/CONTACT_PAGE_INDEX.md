## 🎨 Design System

### Colors
- **TerraVik**: #2d5016 (primary), #7cb342 (secondary), #d4af37 (accent)
- **Hawai Agro**: #1b5e20 (primary), #558b2f (secondary), #81c784 (accent)

### Typography
- Headings: Bold, responsive sizing
- Body: Regular, 14-16px
- Labels: Small, semibold

### Spacing
- Section padding: 3rem 2rem
- Form gap: 1.5rem
- Border radius: 8px (cards, inputs), 6px (buttons)

### Animations
- Form entry: 0.5s fade + slide
- Error/Success: 0.3s fade + slide
- Button hover: 0.3s scale
